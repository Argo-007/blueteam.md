# blueteam.md
herramientas desarrolladas para analisis de blueteam

Esta primer herramienta fue realizada para consultar rapidamente mediante API a virustotal e ipabuse, 
despues realizar una evaluacion de los resultados de ambas consultas.
Se pueden realizar multiples consultas solo agregandolas a un listado

El listado se agrega en un documento txt llamado ips.txt
